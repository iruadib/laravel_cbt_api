## Setup
Untuk menjalankan projek ini, lakukan langkah berikut:
```powershell
$ cd folder
$ npm install
$ npm run prod
$ php artisan serve
```
atau
```powershell
$ cd folder
$ yarn install
$ yarn prod 
$ php artisan serve
```
Untuk melihaat hasilnya dapat mengunjungi [http://127.0.0.1:8000](http://127.0.0.1:8000/) atau [127.0.0.1:8000](http://127.0.0.1:8000/).
